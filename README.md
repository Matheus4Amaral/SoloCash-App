SoloCash

Aplicativo mobile para controle financeiro pessoal, focado em pessoas que moram sozinhas.

Sobre o projeto

O SoloCash tem como objetivo ajudar usuários a organizarem suas finanças mensais de forma simples, permitindo o controle de receitas, despesas e saldo disponível.

O app foi desenvolvido como projeto da disciplina de Dispositivos Móveis.
